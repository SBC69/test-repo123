# Настройка анимации кота

## Шаг 1: Добавьте spritesheet в проект

1. Откройте проект в Godot
2. Создайте папку `assets/sprites` (если ещё нет)
3. Скопируйте файл `cat-1.6.jpg` в `assets/sprites/cat-spritesheet.png`
4. В Godot FileSystem выберите файл и в инспекторе Import установите:
   - **Filter**: `Nearest` (для пиксель-арта)
   - **Mipmaps**: `Off`
   - Нажмите **Reimport**

## Шаг 2: Настройте SpriteFrames

1. Откройте `scenes/Player.tscn`
2. Выберите узел `AnimatedSprite2D`
3. В инспекторе найдите **Sprite Frames**
4. Нажмите на кнопку `[empty]` рядом с Sprite Frames
5. Выберите **New SpriteFrames**
6. Нажмите на созданный SpriteFrames ресурс

## Шаг 3: Добавьте анимации

В панели SpriteFrames (внизу экрана) создайте следующие анимации:

### Анализ spritesheet:
По картинке видно, что:
- **8 кадров по горизонтали** (columns)
- **Много рядов** (каждый ряд = отдельная анимация)

### 1. Анимация "idle" (покой)
1. Нажмите кнопку **Add Animation** в панели SpriteFrames
2. Назовите её `idle`
3. Нажмите кнопку **Add frames from sprite sheet**
4. Выберите `cat-spritesheet.png`
5. В окне настройки:
   - **Horizontal**: `8` (количество кадров в ряду)
   - **Vertical**: `подсчитайте количество рядов`
6. Выделите первый ряд (кот стоит) и нажмите **Add Frames**
7. Установите **FPS**: `8-10`

### 2. Анимация "run" (бег)
1. Создайте новую анимацию `run`
2. Добавьте кадры из ряда с бегущим котом (вероятно, ряд 3-4)
3. **FPS**: `12-15`

### 3. Анимация "jump" (прыжок)
1. Создайте `jump`
2. Добавьте кадры с прыжком (ряд 5-6, кот в воздухе)
3. **FPS**: `10`

### 4. Анимация "fall" (падение)
1. Создайте `fall`
2. Добавьте кадры с падением (часто тот же ряд, что и jump, но последние кадры)
3. **FPS**: `10`

### 5. Анимация "attack" (атака)
1. Создайте `attack`
2. Добавьте кадры с атакой (кот царапается или бьёт лапой)
3. **FPS**: `15`
4. **Loop**: `Off` (чтобы анимация не зацикливалась)

### 6. Анимация "death" (смерть)
1. Создайте `death`
2. Добавьте кадры со смертью (кот падает или исчезает)
3. **FPS**: `8`
4. **Loop**: `Off`

## Шаг 4: Настройте размер коллизии

1. Выберите `CollisionShape2D` в Player
2. Настройте размер **RectangleShape2D** под размер кота
3. Рекомендуемый размер: `Vector2(24, 32)` или подберите визуально

## Шаг 5: Запустите игру

Нажмите **F5** или кнопку Play. Теперь ваш персонаж должен быть анимированным котом!

## Что изменено в коде:

- Заменили `ColorRect` на `AnimatedSprite2D`
- Добавили метод `_update_animation()`, который переключает анимации
- Используем `flip_h` для отзеркаливания
- Анимации переключаются автоматически в зависимости от состояния (idle, run, jump, fall, attack, death)
