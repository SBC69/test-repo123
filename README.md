# Спасение из подземелья

Простая 2D игра-платформер на Godot Engine 4.2

## Сюжет

Рыжего котика бросили в подземелья злого крысиного короля. Он должен выбраться оттуда, победить крысиного короля и спасти трёхцветную пушистую кошечку - свою любовь.

## Управление

- **A / Стрелка влево** - движение влево
- **D / Стрелка вправо** - движение вправо
- **W / Пробел** - прыжок
- **F** - атака

## Структура проекта

```

```

## Особенности

### Игрок (Player.gd)
- Движение влево/вправо
- Прыжки
- Атака ближнего боя
- Система здоровья (100 HP)
- Анимации: idle, run, jump, attack, hurt, death

### Враги (Enemy.gd)
- Патрулирование территории
- Обнаружение игрока
- Преследование
- Атака при приближении
- 50 HP здоровья

### Босс (Boss.gd)
- Крысиный король с 200 HP
- Две фазы боя (вторая активируется при 50% здоровья)
- Обычная атака (25 урона)
- Специальная атака - прыжок сверху (40 урона)
- Полоса здоровья

### Принцесса (Princess.gd)
- Трёхцветная кошечка
- При спасении запускается победный экран

## Как запустить

1. Установите Godot Engine 4.2 или новее
2. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/SBC69/test-repo123.git
   ```
3. Откройте проект в Godot
4. Создайте необходимые сцены в папке `scenes/`:
   - MainMenu.tscn
   - Level1.tscn
   - Level2.tscn (опционально)
   - BossLevel.tscn
   - Victory.tscn
   - GameOver.tscn
5. Добавьте спрайты и анимации для персонажей
6. Настройте физические слои и коллизии
7. Запустите игру (F5)
